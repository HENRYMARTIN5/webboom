WebBoom is a  WebAssembly port of Doom based on PrBoom+.

You can load a custom WAD using the button below. To use it, modify the command (i.e. adding  -file you_file.wad) before clicking on the "Launch Doom" button.

WebDoom comes with the  shareware version of Doom preloaded, but it also supports The Ultimate Doom and Doom II, and also any WAD based off these two. This link contains a list of games and where to get them, but keep in mind that games based on Heretic or Hexen won't work.
